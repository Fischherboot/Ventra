# Project Ventra – Tablet Erklär-App

**Autor:** Moritz / Moritzsoft

Eine First-Time-Use-App für Android-Tablets, die absoluten Anfängern Schritt für Schritt erklärt, wie ein Tablet funktioniert.

## Anforderungen

- Android Studio (neueste Version)
- Android SDK 35
- Min SDK: 29 (Android 10)
- Gradle 8.10.2
- Kotlin 2.0.21

## Projekt öffnen

1. Android Studio öffnen
2. **File → Open** → diesen Ordner auswählen
3. Gradle synchronisieren lassen
4. Auf ein 10-Zoll-Tablet deployen

## Tutorial-Ablauf

| State | Beschreibung |
|-------|-------------|
| 0 | Begrüßung durch Sophie |
| 1 | Name eingeben |
| 2 | Tastatur üben ("Super!" tippen) |
| 3 | Ein-/Aus-Knopf ausprobieren |
| 4 | Lautstärke-Wippe testen |
| 5 | Tablet in die Ladestation legen |
| 6 | Abschluss |

## Hinweise

- Die App läuft **nur im Querformat** (Landscape)
- Optimiert für **10-Zoll-Tablets**
- Die Musik im Lautstärke-Test wird **programmatisch erzeugt** (keine externe Datei nötig)
- Kein Fortschritt wird gespeichert – bei jedem Start beginnt das Tutorial von vorne
